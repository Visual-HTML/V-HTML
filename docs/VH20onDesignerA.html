Welcome to Macromedia Extension Manager 1.7

The Extension Manager lets you easily install new extensions and keep track of new extensions you download from the Macromedia Exchange site. It also helps you share extensions by automatically packaging and submitting the extensions you create.

We have some great resources to help you discover more about Macromedia Extension Manager, get support, and send feedback to the team.


To Get Started
To launch the Extension Manager in Dreamweaver, Fireworks, or Flash, choose Help > Manage Extensions.

Release Notes
Release Notes contain late-breaking information and known issues about the Extension Manager.

Macromedia Exchange
Download extensions to customize your Macromedia products.

The Macromedia Extension Installation File Format describes how to create an extension package that you can share with other users.

You can share an extension that you have created by putting it on the Macromedia Exchange. Testing guidelines and submission steps are provided within the Submit to the Exchange help document.

Dreamweaver Extensibility Newsgroup is for JavaScript and C developers who are creating objects, behavior actions, commands, property inspectors, floating palettes, and translators for Dreamweaver to discuss the extensibility mechanism and coding techniques.
